# linux-server-infrastructure
🖥️ Linux Server Infrastructure Project

This project demonstrates a complete Linux server setup covering System Administration, Networking, and Security fundamentals.

🚀 Project Overview

In this project, I built a Linux-based server environment that includes:

Web Server hosting multiple websites
File sharing between Linux and Windows
User and permission management
Firewall configuration
Basic automation using Bash scripting
🔧 Technologies Used
Linux (Debian/Ubuntu)
Apache / Nginx
Samba
UFW Firewall
Bash Scripting

🌐 Features
1. Web Server
Configured multiple virtual hosts
Hosted multiple websites on the same server
2. File Sharing
Implemented Samba for file sharing
Accessed shared files from Windows client
3. User Management
Created users and groups
Applied proper permissions and ownership
4. Security
Configured UFW firewall
Allowed only necessary ports (22, 80, 443)
5. Automation
Created a backup script using Bash
Automated tasks using cron jobs

🌍 Networking
Worked with Linux network interfaces (default configuration)
Verified IP addressing and connectivity
Ensured services are accessible over the network
